# Jogo Bonito

## Derivation
"Jogo Bonito" is Portuguese for "The Beautiful Game," a phrase immortalized by Pelé. No team bias, no nation bias — just football intelligence for everyone, running entirely on your local device.

## Prerequisites
- **Node.js**: Version $\ge$ 22.17
- **npm** (included with Node.js)

## Installation
```bash
npm install
```

## Running the Application
```bash
npm start
```

> [!NOTE]
> **First Run Note:** On the first execution, the application will automatically download the local LLM model file. Subsequent launches will use the cached local model file for instant, completely offline startup.

## Features

### ⚽ The Six Modes

1. **Tactician**
   Expert football tactician mode. Provide any tactical prompt, formation debate, or squad query to get deep tactical reports on systems, player roles, and transition phases.
   
2. **Learn**
   Football conceptual tutor. Analyzes the complexity of your first message to dynamically set your knowledge profile (**Beginner**, **Intermediate**, or **Expert**) and adjust explanation depth accordingly.

3. **Predict**
   Match prediction center. Select Home and Away squads to load side-by-side QVAC stats (Quality, Value, Cohesion, Variation) and generate detailed local match forecasts.

4. **Debate**
   Head-to-head football debate. Input opposing views (e.g. Messi vs. Ronaldo or High Press vs. Low Block) to trigger an offline debate, rendering a structured visual Verdict Card with a clear Winner, Key Arguments, and a Confidence score.

5. **Commentate**
   Timeline commentary engine. Click match event buttons (KICKOFF, GOAL, FOUL, substitution, etc.) to build a dynamic timeline ticker. The local AI generates custom commentary scripts, and the application plays each line spoken aloud sequentially.

6. **Formation Builder**
   Drag-and-drop playground. Select one of the 13 international or club football squads (equipped with full 11-player rosters) and arrange players visually onto an interactive pitch representation. Analyze your custom formation alignment against the squad's actual manager setup.
